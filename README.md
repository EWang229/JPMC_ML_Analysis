# Income Classification and Customer Segmentation Project

## Overview

This project develops a machine learning pipeline for a retail business client to:
1. Predict whether an individual earns more than $50K annually (binary classification)
2. Segment individuals into meaningful customer groups using unsupervised learning

The workflow includes data cleaning, feature engineering, exploratory data analysis, supervised modeling (Logistic Regression), and unsupervised clustering (KMeans).
